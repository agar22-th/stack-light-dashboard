# 🚦 Stack Light IIoT System (PLC → Grafana)

This project demonstrates a complete Industrial IoT (IIoT) monitoring system, from a Siemens PLC to real-time data visualization.

## 📌 Overview

A 3-LED Stack Light (Red, Yellow, Green) is controlled using a Siemens S7-1500 PLC and monitored through a full data pipeline including Node-RED, InfluxDB, and Grafana.

---

## ⚙️ System Architecture

PLC (S7-1500) → Modbus TCP → Node-RED → InfluxDB → Grafana → Excel/CSV

---

## 🔧 Technologies Used

- Siemens **S7-1500 PLC**
- **TIA Portal**
- **Modbus TCP (Port 502)**
- **Node-RED**
- **InfluxDB**
- **Grafana**
- **Excel / CSV Export**

---

## 🚀 Features

- Real-time control of Stack Light (Red / Yellow / Green)
- Data exchange via Modbus TCP
- Live dashboard using Node-RED
- Time-series data storage with InfluxDB
- Advanced visualization using Grafana
- Data export to Excel/CSV
